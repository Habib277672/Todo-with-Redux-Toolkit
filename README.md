📌 Todo App with Redux Toolkit

A modern, responsive Todo app built with HTML, CSS, React.js, and Redux Toolkit.
This project demonstrates state management using Redux Toolkit, functional React components, and clean UI design.

👉 Live Demo: https://habib277672.github.io/Todo-with-Redux-Toolkit/

🧠 Features

✅ Add new todos

❌ Delete todos

✏️ Mark todos as complete/incomplete

🗂️ State managed using Redux Toolkit

🧩 Fully responsive UI

🛠️ Tech Stack
Layer Technology
UI HTML, CSS
Frontend React.js
State Management Redux Toolkit
Hosting GitHub Pages
